# Python Analyst Project

## 📌 Overview

This repository contains Python-based data analysis scripts and workflows. It is designed for learning, experimentation, and showcasing data analyst skills.

---

## 🚀 Getting Started

### 1. Clone the Repository

Open your terminal (or VS Code terminal) and run:

```bash
git clone https://github.com/tadde201/Python_Analyst.git
cd Python_Analyst
```

---

### 2. Open in VS Code

```bash
code .
```

---

### 3. (Optional) Set Up Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate   # On Windows
```

---

### 4. Install Dependencies

If you have a requirements file:

```bash
pip install -r requirements.txt
```

---

## 🔄 Git Workflow

### Check status

```bash
git status
```

### Add changes

```bash
git add .
```

### Commit changes

```bash
git commit -m "Describe your changes"
```

### Push to GitHub

```bash
git push origin main
```

---

## 🌿 Branching Strategy

### Create a new branch

```bash
git checkout -b feature-name
```

### Push branch

```bash
git push origin feature-name
```

### Merge into main

```bash
git checkout main
git merge feature-name
git push origin main
```

---

## 🔁 Keep Your Code Updated

```bash
git pull origin main
```

---

## 🧠 Best Practices

* Use meaningful commit messages
* Create branches for new features or fixes
* Pull latest changes before pushing
* Avoid working directly on `main` in team environments

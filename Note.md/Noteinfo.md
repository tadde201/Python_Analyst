File & Folder Navigation
pwd              # Show current directory
ls               # List files
ls -la           # List all files (including hidden)
cd folder_name   # Go into a folder
cd ..            # Go back one level
mkdir new_folder # Create folder
rm file.txt      # Delete file
rm -r folder     # Delete folder
Clone repo
git clone https://github.com/tadde201/Python_Analyst.git
Check status
git status
Add & commit
git add .
git commit -m "your message"
Push changes
git push origin main
Pull latest code
git pull origin main
🌿 Branching (Interview Critical)
git checkout -b new-branch     # Create branch
git checkout main             # Switch branch
git merge new-branch          # Merge branch
git branch                    # List branches
🔍 Useful Dev Commands
clear            # Clear terminal
history          # Show command history
echo "text"      # Print text
cat file.txt     # View file content
🐍 Python Commands
python file.py          # Run Python script
python -m venv venv     # Create virtual environment
venv\Scripts\activate   # Activate (Windows)
pip install package     # Install package
pip list                # Show installed packages
⚡ Power Commands (Advanced / Pro)
code .                  # Open project in :contentReference[oaicite:1]{index=1}
git log                 # View commit history
git diff                # See changes
git stash               # Save work temporarily
git reset --hard        # Reset changes (⚠️ destructive)
